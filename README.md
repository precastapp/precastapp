# Precastapp

A template for fast build app using Flutter.

## Usage

```bash
dart run melos bs
cd app
flutter run
```

## Architecture

The app is built using Flutter and Melos.
Structure of directories:

```
├── app
├── core
├── design_system
├── modules
│   ├── auth
│   └── home
│   └── increment_sample
│   └── people
└── README.md
```
